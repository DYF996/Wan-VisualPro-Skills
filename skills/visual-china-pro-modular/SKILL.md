```
┌─────────────────────────────────────────────┐

├─────────────────────────────────────────────┤
│  ① 我想让这张图看起来：                       │

│                                              │
│  ③ 有什么特别想突出的元素吗？                   │
│     [产品/人/氛围/色彩/其他]                  │
│                                              │
│  ④ 有参考风格吗？                             │
│     [上传参考图 / 描述参考风格]               │
│                                              │
│  ⑤ 有哪些必须避免的元素？                      │
│     [过度美化/失真/太暗/其他]                  │
└─────────────────────────────────────────────┘
```

### 步骤2：场景与模块推荐

**📝 操作：** 根据用户输入，智能推荐：

```
推荐场景：[场景编号+名称] - 匹配度XX%
推荐光影：[A1-A6]
推荐色彩：[B1-B8]  
推荐质感：[C1-C6]
推荐构图：[D1-D6]

⚠️ 是否接受推荐配置？
→ [确认] → 直接使用推荐配置
→ [微调] → 进入步骤3
→ [自定义] → 完全自定义
```

### 步骤3：参数微调

**📝 操作：** 用户可选择以下调整方式：

```
【快速调整】- 从预设方案中选择
  光影：[A1] [A2] [A3] [A4] [A5] [A6]
  色彩：[B1] [B2] [B3] [B4] [B5] [B6] [B7] [B8]
  质感：[C1] [C2] [C3] [C4] [C5] [C6]
  构图：[D1] [D2] [D3] [D4] [D5] [D6]

【精细调整】- 输入具体参数
  光影色温：[XXXXK]
  色彩饱和度：[±XX%]
  质感强度：[XX%]
  构图比例：[X:X]
```

### 步骤4：智能拼接

**📝 操作：** 将所有参数智能拼接为万相模型可识别的Prompt

### 步骤5：生成与预览

**📝 操作：** 调用万相模型，生成4个变体供选择

```
生成4个变体：
┌──────────┬──────────┐
│  变体A    │  变体B   │
│  标准推荐 │  色彩强化 │
├──────────┼──────────┤
│  变体C    │  变体D   │
│  质感强化 │  创意版本 │
└──────────┴──────────┘

每个变体附带：
- 4维评分（商业传达力/视觉冲击力/技术完成度/创新性）
- 与用户原始意图的匹配度
```

### 步骤6：精修与交付

**📝 操作：** 用户选择后进行微调，输出最终成品
